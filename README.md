# NoteNest

軽量なプロジェクト型ノート管理ツール（Windows デスクトップアプリ）

## 概要

NoteNest は、複数のノートブックとノート、プロジェクト全体のタスク、本文中のマーカーを **1 つのプロジェクトファイル** にまとめて管理する Windows デスクトップアプリです。

単純な Markdown ファイルエディタではなく、プロジェクト単位で情報を統合管理することを目的としています。

## v0.1.0 の機能

### ノート管理
- ノートブック／ノートのツリー表示（左ペイン）
- ノートブックの追加・名前変更・削除
- ノートの追加・名前変更・削除
- ノート本文の編集（標準テキストエディタ）
- 右端折り返し・縦スクロール

### 検索と置換
- キーワード検索（次を検索）
- 置換・すべて置換
- 大文字／小文字の区別指定

### 表示設定
- フォント種類の変更（日本語フォント対応）
- フォントサイズの変更

### タスク管理（右ペイン上段）
- 今日のタスク／今週のタスク／バックログへの分類
- タスクの追加・名前変更・削除
- チェックボックスによる完了状態の切り替え
- 各グループの件数表示

### マーカー（右ペイン下段）
- 本文中の `[TODO]`・`[FIXME]`・`[NOTE]` を自動抽出
- 行番号・ノート名・抜粋の表示
- クリックで該当行へジャンプ
- **v0.1.1** 利用可能なマーカー記法を「？」アイコンのTooltipで確認可能
- **v0.1.1** プロジェクト全体のマーカー集計を右下ペイン最下部に表示（全ノートを対象）
- **v0.1.1** マーカーを含むノートは左ペインに「●」マークで表示

### ファイル管理
- プロジェクトを `.notenest` ファイルとして保存・読込
- 新規プロジェクト作成（サンプルデータ付き）
- 未保存変更の確認ダイアログ
- 最後に開いていたノートの自動復元

## 起動方法

### 前提条件
- Windows 10 / 11
- .NET 8.0 Runtime（Windows デスクトップランタイム）

### ビルドと実行
```
git clone <repository-url>
cd notenest
dotnet build NoteNest/NoteNest.csproj -c Release
dotnet run --project NoteNest/NoteNest.csproj
```

または Visual Studio 2022 でソリューションファイル `NoteNest.sln` を開いて実行してください。

## `.notenest` ファイルについて

- 拡張子：`.notenest`
- 形式：UTF-8 エンコードの JSON
- 保存内容：プロジェクト名・ノートブック・ノート本文・タスク・表示設定

```json
{
  "version": "0.1.0",
  "projectId": "...",
  "projectName": "業務改善プロジェクト",
  "notebooks": [...],
  "tasks": { "today": [...], "week": [...], "backlog": [...] },
  "settings": { "lastOpenedNoteId": "...", "fontFamily": "Yu Gothic UI", "fontSize": 14 }
}
```

## キーボードショートカット

| キー | 操作 |
|------|------|
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新規プロジェクト |
| `Ctrl+O` | プロジェクトを開く |
| `Ctrl+F` | 検索／置換ダイアログを開く |
| `Ctrl+H` | 検索／置換ダイアログを開く |

## 既知の制限（v0.1.1）

- Markdown のプレビューおよびシンタックスハイライトは未実装
- 行番号表示は未実装
- マーカーはプロジェクト全体の集計値のみ表示（全ノート横断のクリックナビゲーションは未対応）
- タスクへの期限・優先度・担当者の設定は未実装
- 複数プロジェクトの同時管理は未対応
- 添付ファイル管理は未実装
- 自動保存は未実装（手動保存のみ）
