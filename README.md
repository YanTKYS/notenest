# NoteNest

軽量なプロジェクト型ノート管理ツール（Windows デスクトップアプリ）

## 概要

NoteNest は、複数のノートブックとノート、プロジェクト全体のタスク、本文中のマーカーを **1 つのプロジェクトファイル** にまとめて管理する Windows デスクトップアプリです。

単純な Markdown ファイルエディタではなく、プロジェクト単位で情報を統合管理することを目的としています。

## v0.1.3 の追加機能

### タスクのグループ間移動
- タスクを右クリック →「グループを変更」から別グループへ移動できます
- 「今日のタスク」「今週のタスク」「バックログ」間を移動可能

### 完了済みタスクの非表示
- グループヘッダーの「完了非表示」チェックボックスで完了タスクを非表示にできます
- グループごとに独立して設定可能

### タスクコメント編集のヒント
- タスクタイトルにマウスオーバーすると「ダブルクリックでコメントを追加/編集」と表示されます

## v0.1.2 の追加機能

### 複数プロジェクトの同時利用
- NoteNest を複数起動できます（多重起動を許可）
- それぞれ別の `.notenest` ファイルを開いて同時利用可能
- **注意**：同じ `.notenest` ファイルを複数ウィンドウで同時編集すると、後から保存した内容で上書きされます

### ファイルなし起動時の導線
- ファイルを指定せずに起動すると、サンプルプロジェクトを表示
- 中央エディタ上部に案内バナーを表示（「サンプルプロジェクトです。保存してプロジェクトを開始できます」）
- 保存後はバナーが自動的に消えます

### マーカー挿入ボタン
- 中央エディタ下部に `[TODO]` `[FIXME]` `[NOTE]` の挿入ボタンを追加
- ボタンをクリックすると、カーソル位置にマーカー記法を挿入
- 挿入後、右下マーカー一覧と集計値が更新されます

### タスクコメント
- タスクごとにコメントを記入できます
- タスクタイトルをダブルクリックすると、中央エディタでコメントを編集
- コメントは `.notenest` ファイルに保存されます
- ノートを選択すると、通常のノート編集に戻ります

### コメント付きタスクの視認性
- コメントが設定されているタスクには「●」マーク（青）を表示
- マークにマウスオーバーすると「このタスクにはコメントがあります」を表示

### v0.1.2 では実装しないこと
- 画像貼り付け（NoteNest は軽量テキスト管理ツールのため）
- 共同編集（ローカル単一ファイル管理の思想と合わないため）
- 文字数表示
- マーカー行の表示／非表示（WPF 標準 TextBox では本文消失リスクがあるため）

## v0.1.0 の機能

### ノート管理
- ノートブック／ノートのツリー表示（左ペイン）
- ノートブックの追加・名前変更・削除
- ノートの追加・名前変更・削除
- ノート本文の編集（標準テキストエディタ）
- 右端折り返し・縦スクロール

### 検索と置換
- キーワード検索（次を検索）
- 置換・すべて置換
- 大文字／小文字の区別指定

### 表示設定
- フォント種類の変更（日本語フォント対応）
- フォントサイズの変更

### タスク管理（右ペイン上段）
- 今日のタスク／今週のタスク／バックログへの分類
- タスクの追加・名前変更・削除
- チェックボックスによる完了状態の切り替え
- 各グループの件数表示

### マーカー（右ペイン下段）
- 本文中の `[TODO]`・`[FIXME]`・`[NOTE]` を自動抽出
- 行番号・ノート名・抜粋の表示
- クリックで該当行へジャンプ
- **v0.1.1** 利用可能なマーカー記法を「？」アイコンのTooltipで確認可能
- **v0.1.1** プロジェクト全体のマーカー集計を右下ペイン最下部に表示（全ノートを対象）
- **v0.1.1** マーカーを含むノートは左ペインに「●」マークで表示

### ファイル管理
- プロジェクトを `.notenest` ファイルとして保存・読込
- 新規プロジェクト作成（サンプルデータ付き）
- 未保存変更の確認ダイアログ
- 最後に開いていたノートの自動復元

## 起動方法

### 前提条件
- Windows 10 / 11
- .NET 8.0 Runtime（Windows デスクトップランタイム）

### ビルドと実行
```
git clone <repository-url>
cd notenest
dotnet build NoteNest/NoteNest.csproj -c Release
dotnet run --project NoteNest/NoteNest.csproj
```

または Visual Studio 2022 でソリューションファイル `NoteNest.sln` を開いて実行してください。

## `.notenest` ファイルについて

- 拡張子：`.notenest`
- 形式：UTF-8 エンコードの JSON
- 保存内容：プロジェクト名・ノートブック・ノート本文・タスク・表示設定

```json
{
  "version": "0.1.0",
  "projectId": "...",
  "projectName": "業務改善プロジェクト",
  "notebooks": [...],
  "tasks": { "today": [...], "week": [...], "backlog": [...] },
  "settings": { "lastOpenedNoteId": "...", "fontFamily": "Yu Gothic UI", "fontSize": 14 }
}
```

## キーボードショートカット

| キー | 操作 |
|------|------|
| `Ctrl+S` | 保存 |
| `Ctrl+N` | 新規プロジェクト |
| `Ctrl+O` | プロジェクトを開く |
| `Ctrl+F` | 検索／置換ダイアログを開く |
| `Ctrl+H` | 検索／置換ダイアログを開く |

- 添付ファイル管理は未実装
- 自動保存は未実装（手動保存のみ）
