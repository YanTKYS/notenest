================================================================
NestSuite .NET Framework 4.8 テストビルド
[軽量化検証用 / 正式サポート外 / Experimental]
================================================================

このバイナリは NestSuite の .NET Framework 4.8 前提テストビルドです。
正式リリースは同梱の nestsuite_*.zip（.NET 8 自己完結版）を使用してください。

----------------------------------------------------------------
目的
----------------------------------------------------------------
.NET Framework 4.8 が既にインストールされた環境で NestSuite が
framework-dependent（ランタイム非同梱）として起動できるかを
実機検証するための実験的ビルドです。

通常運用・本番環境での使用は想定していません。

----------------------------------------------------------------
前提条件
----------------------------------------------------------------
- Windows 10 以降（推奨）
- .NET Framework 4.8 ランタイムがインストール済みであること

  未インストールの場合は以下からダウンロードしてください:
  https://dotnet.microsoft.com/download/dotnet-framework/net48

----------------------------------------------------------------
実行方法
----------------------------------------------------------------
1. ZIP を展開してください
2. NestSuite.exe を直接実行してください
3. .NET Framework 4.8 ランタイムが読み込まれて起動します

----------------------------------------------------------------
注意事項
----------------------------------------------------------------
- 正式サポート外・バグ報告の対象外です
- 動作が不安定な場合は .NET 8 版（nestsuite_*.zip）を使用してください
- 本ビルドのデータは .NET 8 版と同じ保存形式（スキーマ 1.4.1）を
  使用するため、ファイルの互換性はあります
- このビルドは自己完結型（self-contained）ではありません
  .NET Framework 4.8 DLL はシステムの GAC から読み込みます
