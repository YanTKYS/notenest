- [ ] TempNest への変更があっても未保存確認ダイアログが表示されない（終了時・タブクローズ時）

## チェックリスト追加基準

チェック項目を追加・変更する際は以下の基準に従う。

- **常用チェックリストに置くもの**: 毎回のリリースで確認すべき機能・品質確認
- **補足文書（history）へ置くもの**: 一時的な注意、特定バージョンの経緯、背景説明、既知の非ブロッキング警告
- 過去の失敗履歴をそのまま常用項目化しない
- 追加した項目が長文化した場合は、確認文と背景説明を分ける
- 「念のため」の項目を増やしすぎない
- 重要な安全確認が埋もれないことを優先する
