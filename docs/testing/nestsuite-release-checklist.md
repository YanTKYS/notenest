- [ ] IdeaNest タブを新規作成できる

- [ ] ChatNest タブを複数同時に開ける

### ファイルを開く

- [ ] `.notenest` ファイルを開ける

- [ ] `.ideanest` ファイルを開ける

- [ ] 同じ `.ideanest` を再度開くと既存タブがアクティブになる

### 複数ファイル一括オープン（v1.16.0）

- [ ] 「開く...」ダイアログで複数ファイルを同時選択できる
- [ ] `.notenest` / `.chatnest` / `.ideanest` を混在して複数選択し、それぞれタブとして開ける

### 最近使ったファイル（v1.14.0 / v1.14.1）

- [ ] ファイルメニューに「最近使ったファイル」サブメニューが表示される

### 保存

## 4.6. IdeaNest 編集導線確認（v2.1.0 / v2.1.1）

### 詳細画面の基本操作

- [ ] カードをクリックすると詳細画面が開き、タイトル・本文・タグが即座に編集可能な状態で表示される
- [ ] 「✎ 編集」ボタンが表示されない
- [ ] タイトル TextBox に直接入力できる
- [ ] 本文 TextBox に直接入力でき、Enter で改行できる
- [ ] タグ TextBox にカンマ区切りでタグを入力できる
- [ ] カラーピッカー（色丸ボタン）を選択すると背景色が即座に変わる
- [ ] 「📌 ピン留め」ボタンをクリックすると ★ アイコンが表示され、ボタンラベルが「ピン留め解除」に変わる
- [ ] 「📥 アーカイブ」ボタンをクリックするとアーカイブ済みバッジが表示され、ボタンラベルが「アーカイブ解除」に変わる

### 保存タイミング

- [ ] 詳細画面を閉じると編集内容がカードに反映される（カード一覧のタイトル・色等が更新される）
- [ ] 「前へ」ボタンで前のカードへ移動すると、移動前のカードの編集内容が保存される
- [ ] 「次へ」ボタンで次のカードへ移動すると、移動前のカードの編集内容が保存される
- [ ] Ctrl+S を押すと即座に保存される（ウィンドウは閉じない）
- [ ] 何も変更せず閉じた場合、UpdatedAt が更新されない

### キーボード操作

- [ ] Escape キーで詳細画面を閉じる（編集内容は保存される）
- [ ] 本文 TextBox にフォーカスがある状態で ← / → キーを押してもカード移動しない（テキストカーソルが動く）
- [ ] 本文 TextBox にフォーカスがない状態で ← / → キーを押すとカードを移動する

### 旧編集ダイアログへの導線廃止確認（v2.1.1）

- [ ] カードにカーソルを合わせても「✎ 編集」ボタンが表示されない

- [ ] 「＋」ボタンを押すと、旧編集ダイアログではなく詳細画面ベースの新規作成画面が開く
- [ ] 新規作成画面のタイトルが「新規アイデア」になっている
- [ ] 新規作成画面で前へ/次へボタンが無効になっている

### 新規作成確認（v2.1.1）

- [ ] 新規作成画面で日付行（作成日・更新日）が表示されない（v2.1.2）
- [ ] 新規作成画面でタイトル・本文・タグ・色・ピン留め・アーカイブを入力できる
- [ ] 何も入力せず閉じた場合、カードが追加されない
- [ ] 色を選んだだけ・ピン留めを押しただけで閉じた場合、カードが追加されない
- [ ] タイトルだけ入力して閉じると、カードが追加される
- [ ] 本文だけ入力して閉じると、カードが追加される
- [ ] タグだけ入力して閉じると、カードが追加される
- [ ] Ctrl+S を押すとカードが追加される
- [ ] Ctrl+S 後に閉じても同じカードが二重作成されない
- [ ] Ctrl+S 後に追加編集してから閉じると、追加編集分も保存される
