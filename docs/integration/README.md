# docs/integration — ディレクトリ概要

このディレクトリは NestSuite の統合設計に関する文書を管理する。

---

## 文書の分類

| 文書 | 対象バージョン | 分類 | 状態 |
|------|--------------|------|------|
| `nestsuite-preparation.md` | v1.7〜v1.8 | 履歴 | AppShell / Workspace 統合は実装済み |
| `ideanest-save-load-plan.md` | v1.8.2 | 履歴 | IdeaNest 保存・読込は v1.8.3 で実装済み |
| `nestsuite-multi-file-tabs-plan.md` | v1.9.0 | 履歴 | 複数ファイルタブ対応は v1.9.x で実装済み |
| `nestsuite-notenest-multi-file-plan.md` | v1.9.4 | 履歴 | NoteNest 複数ファイルタブ対応は v1.9.5 以降で実装済み |

現行開発で参照すべき文書はこのディレクトリに含まれない。  
現行の設計方針は `docs/development/nestsuite-development-guidelines.md` を参照すること。

---

## 履歴文書として保持する理由

過去の設計判断・却下した案・段階的移行の経緯を記録しており、  
複数ファイルタブの拡張や統合強化を検討する際の参考になる。
