# docs/design — ディレクトリ概要

このディレクトリは NestSuite の設計判断・設計メモを管理する。

## 文書の分類

### 現行参照

| 文書 | 内容 |
|------|------|
| `nestsuite-known-limitations.md` | NestSuite 既知の制約 |
| `design-decisions.md` | 設計判断の背景と理由（v0.2.0 以降の累積記録） |
