## §18 テキストエクスポート（v0.8.1）

### §18-1 プロジェクト全体のテキスト出力

1. ファイルメニュー → エクスポート → プロジェクト全体をテキスト出力... を選択
2. 保存先ファイルと名前を指定して保存する
3. 出力ファイルをメモ帳で開き、以下を確認する
   - プロジェクト名・ノートブック名・ノート名・本文が読める
   - `===` と `---` の区切りが正しく出力されている
   - `[[リンク名]]` や `[TODO]` などの記法がそのまま出力されている
   - 日本語が文字化けしていない
4. キャンセルした場合にファイルが作られないことを確認する

### §18-2 ノートブックごとのテキスト出力

1. ファイルメニュー → エクスポート → ノートブックごとにテキスト出力... を選択
2. 出力先フォルダを選択する
3. 以下を確認する
   - ノートブック数と同じ数の `.txt` ファイルが作成される
   - 各ファイル名がノートブック名（日本語含む）を正しく反映している
4. 完了メッセージが表示されることを確認する
5. キャンセルした場合にファイルが作られないことを確認する

### §18-3 ファイル名安全化・連番付与

1. `プロジェクト/管理` など使用不可文字を含むノートブック名でエクスポートする
2. `プロジェクト_管理.txt` として出力されることを確認する
3. 同名ノートブックが複数ある場合、`同名.txt` と `同名_2.txt` が作成されることを確認する

### §18-4 エラー処理

1. 書き込み権限がないフォルダへの出力でエラーメッセージが表示されることを確認する
2. ノートブックが 0 件の状態でノートブックごと出力を試みると「エクスポートするノートブックがありません」と表示されることを確認する

### 対象外機能の非実装確認

| 確認項目 | 期待結果 |
|----------|---------|
| 本文の `[TODO]` を入力しても右ペインのタスクは自動作成されない | タスク一覧に変化なし |
| 画像ファイルをエディタにドラッグしても貼り付けられない | 何も起きないまたは無視される |
| マーカー一覧はノート本文からのみ抽出される（タスクコメントは対象外） | コメント内のマーカーは一覧に表示されない |
| 保存した `.notenest` ファイルにマーカー専用フィールドが存在しない | JSON にマーカーデータなし |
