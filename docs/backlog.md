# Backlog — NestSuite

## 1. この文書の役割

`docs/backlog.md` は以下のみを管理する。

- **未着手・保留・将来候補**の実装候補
- **長期構想・保留**（`LT-` prefix）
- **見送り・採用しない方針**（`RJ-` prefix）

この文書が管理しないもの:

- **完了済み項目** → `docs/release-notes.md` で管理する
- 取り消し線で残した完了項目（積み上げない）
- `<details>完了済み</details>` セクション（作成しない）

完了した項番は **欠番** として扱い、再利用しない。  
実装履歴を確認する場合は `docs/release-notes.md` を検索する。

## 2. 採番・追記ルール

### prefix 一覧

| prefix | 対象セクション | 例 |
|--------|----------------|----|
| `SH-` | NestSuite Shell 改善 | `SH-1` |
| `TN-` | TempNest 改善 | `TN-3` |
| `L` / `M` / `H` | NoteNest Workspace 改善（難易度別） | `L4` / `M2` / `H3` |
| `ID-` | IdeaNest Workspace 改善 | `ID-4` |
| `CH-` | ChatNest Workspace 改善 | `CH-11` |
| `LK-` | タブ間連携 | `LK-1` |
| `FM-` | ファイル形式・マイグレーション | `FM-2` |
| `TD-` | 技術的負債・保守性 | `TD-33` |
| `LT-` | 長期構想・保留 | `LT-1` |
| `RJ-` | 見送り・採用しない方針 | `RJ-1` |

### 必須ルール

- 新規項目は該当 prefix のセクション末尾に追加する
- 完了した項目は backlog から削除し、`docs/release-notes.md` に記録する
- 完了済み項番は再利用しない（欠番として扱う）
- 取り消し線で完了項目を残さない
- `<details>完了済み</details>` を追加しない
- 複数体系にまたがる項目は主責務のセクションへ置き、関連 ID を概要に記載する
- 長期構想・保留は `LT-` 項番で管理する
- 見送り・採用しない方針は `RJ-` 項番で管理する
- `LT-` / `RJ-` も採番済み番号は再利用しない
- 完了時は `docs/release-notes.md` に backlog ID と実装内容を記録する

### 優先度の定義

- **A**：現行運用上の体感改善が大きく、既存設計への影響が小さい
- **B**：有益だが、設計整理や複数 Workspace への影響確認が必要
- **C**：長期的に有望だが、現時点では慎重に扱う

NoteNest Workspace の改善では「WPF 標準 TextBox の範囲内かどうか」を追加判断基準として参照すること。

## 12. 見送り・採用しない方針（RJ-）

現時点で採用しない方針を明示し再提案を防ぐための記録。  
将来条件が変わった場合は RJ 項目を直接実装候補にせず、新規の該当 prefix 項目を採番する。  
`RJ-` 番号は再利用しない。

| No | 方針 | 概要 | 見送り理由 |
|----|------|------|------------|
| RJ-1 | .NET Framework 4.8 版の正式採用見送り | v2.7.0〜v2.7.2 の `net48_test` 検証で実機起動は確認できたが複数 DLL 構成となる。`NestSuite.Net48Test/` は検証履歴として保持するが通常ビルド・リリースからは参照しない | 単一 EXE 配布方針（RJ-5）と衝突。Costura.Fody / ILRepack 等の単一 EXE 化手段が確立された場合のみ再検討 |
| RJ-2 | 外部クラウド / 外部 API 前提の機能 | AI 要約・自動タグ付け・クラウド同期・CRDT 共同編集など外部通信を前提とする機能全般 | ローカルファースト・閉域方針と衝突する |
| RJ-3 | 保存形式を安易に変更する案 | スキーマ設計・互換読み込み・マイグレーション方針なしで既存保存形式を変更することは行わない | 後方互換・ユーザーデータ保護のため。変更する場合は FM-1 方針（`docs/architecture/schema-versioning-policy.md`）に従うこと |
